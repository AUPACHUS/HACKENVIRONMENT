Topologías de Red

Configuraciones físicas y lógicas para implementación de redes

¿Qué es una Topología de Red?

La topología de red define cómo se interconectan los dispositivos en una red, tanto física como lógicamente. Determina:

  • Flujo de datos
  • Escalabilidad
  • Tolerancia a fallos
  • Costo de implementación
Tip: Las redes modernas suelen combinar múltiples topologías (híbridas).

Topologías Físicas

Estrella

Dispositivo central: Switch o Hub

  • Ventajas: Fácil administración, fallos localizados
  • Desventajas: Dependencia del nodo central
  • Usos: Redes LAN modernas, WiFi

Bus

Cable backbone: Coaxial (10Base2)

  • Ventajas: Bajo costo, simple instalación
  • Desventajas: Un fallo afecta toda la red, colisiones
  • Usos: Redes antiguas, industriales

Anillo

Flujo unidireccional: Token passing

  • Ventajas: Sin colisiones, rendimiento predecible
  • Desventajas: Un fallo interrumpe toda la red
  • Usos: FDDI, redes MAN

Malla

Interconexión total/parcial

  • Ventajas: Redundancia, alta disponibilidad
  • Desventajas: Alto costo, complejidad
  • Usos: Datacenters, redes WAN

Híbrida

Combinación de topologías

  • Ventajas: Flexibilidad, optimización
  • Desventajas: Diseño complejo
  • Usos: Redes empresariales modernas

Topologías Lógicas

Broadcast

  • Todos los nodos reciben los datos
  • Ejemplo: Ethernet (CSMA/CD)
  • Uso en redes locales
Switch > flood (broadcast)
ARP requests

Token Passing

  • Nodo transmite solo con "token"
  • Ejemplo: Token Ring, FDDI
  • Sin colisiones
Nodo A → Token → Nodo B → Datos → Nodo C

Comparativa Técnica

Topología Costo Escalabilidad Tolerancia a Fallos Rendimiento
Estrella Moderado Alta Media (depende del centro) Alto (full-duplex)
Bus Bajo Baja Baja Bajo (colisiones)
Anillo Moderado Media Baja (sin redundancia) Moderado
Malla Alto Alta Muy Alta Óptimo

Casos de Uso Reales

Pequeña Oficina (SOHO)

Topología: Estrella

Dispositivos: 1 Switch + Access Point WiFi

Ventaja: Simpleza y bajo mantenimiento

Campus Universitario

Topología: Híbrida (Estrella + Malla)

Dispositivos: Múltiples switches enlazados

Ventaja: Balance entre costo y redundancia

Data Center

Topología: Malla completa

Tecnología: Spine-and-Leaf

Ventaja: Máxima disponibilidad
