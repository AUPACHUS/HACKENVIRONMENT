Mapas de Calor WiFi

Interpretación de Mapas de Calor

Los mapas de calor visualizan la cobertura de señal en un espacio físico. Ekahau y otros herramientas profesionales permiten:

  • Identificar zonas muertas (sin cobertura)
  • Detectar interferencias entre canales
  • Optimizar ubicación de puntos de acceso
  • Planear redes para alta densidad de dispositivos

Umbrales Recomendados para Redes Empresariales

Métrica Óptimo Aceptable Crítico Herramienta de Medición
RSSI > -55 dBm -55 a -70 dBm < -70 dBm Ekahau, NetSpot, inSSIDer
SNR > 30 dB 20-30 dB < 20 dB Wireshark, WiFi Analyzer
Pérdida de Paquetes < 1% 1-3% > 3% Ping, iPerf
Interferencia Co-Canal < 5% 5-15% > 15% Ekahau, Chanalyzer

Configuración de Repetidores

Atención: Los repetidores reducen el ancho de banda a la mitad. En entornos empresariales, preferir soluciones con cableado backhaul.

Mejores Prácticas para Repetidores

  1. Ubicar en zona con al menos -60 dBm de señal del AP principal
  2. Configurar en canal diferente al AP principal (evitar interferencia)
  3. Usar el mismo SSID pero diferente canal para roaming asistido
  4. Verificar que soporta el mismo estándar WiFi que los APs principales
  5. Priorizar bandas de 5GHz para mayor velocidad

Comandos para Diagnóstico WiFi en Linux

# Escanear redes WiFi disponibles
sudo iwlist wlan0 scan

# Ver información de la conexión actual
iwconfig wlan0

# Test de velocidad (requiere iperf3 en ambos extremos)
iperf3 -c [server_ip] -t 20 -i 2

# Medir potencia de señal en tiempo real
watch -n 1 "iwconfig wlan0 | grep -i quality"

# Analizar espectro WiFi (requiere tarjeta compatible)
sudo horst -i wlan0
