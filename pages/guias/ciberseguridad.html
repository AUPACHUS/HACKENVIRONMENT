Fundamentos de Ciberseguridad

Principios esenciales para proteger sistemas y redes

¿Qué es la Ciberseguridad?

La ciberseguridad es la práctica de proteger sistemas, redes y datos de ataques digitales. Abarca:

Confidencialidad

Solo usuarios autorizados acceden a la información.

Integridad

Los datos no son alterados sin autorización.

Disponibilidad

Los sistemas están accesibles cuando se necesitan.

Técnicas de Defensa

Contra Malware

  • Antivirus/EDR: Soluciones como CrowdStrike o Windows Defender.
  • Listas de ejecución: Whitelisting de aplicaciones permitidas.
  • Parches: Mantener sistemas actualizados.

Contra Phishing

# Señales de correos phishing:
1. Remitente sospechoso (ej: soporte@servici0.com)
2. Urgencia ("Su cuenta será bloqueada")
3. Enlaces acortados o dominios extraños
4. Archivos adjuntos inesperados (.exe, .js)

Contra DoS/DDoS

  • Rate limiting: En firewalls y servidores web.
  • CDN: Usar Cloudflare o Akamai para filtrar tráfico.
  • Redundancia: Balanceadores de carga y múltiples servidores.

Modelos de Seguridad

CIA Triad

  • Confidentiality: Cifrado, controles de acceso
  • Integrity: Firmas digitales, hashes
  • Availability: Redundancia, backups

Zero Trust

  • "Nunca confíes, siempre verifica"
  • Microsegmentación
  • Autenticación continua

Ejemplo de Configuración Segura

# Configuración básica de iptables (Linux)
iptables -A INPUT -p tcp --dport 22 -j ACCEPT  # Permitir SSH
iptables -A INPUT -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT
iptables -P INPUT DROP  # Denegar todo lo demás
iptables -P FORWARD DROP
iptables -P OUTPUT ACCEPT
